This is a website I made during a class on typography. The website is about a fontface that was appointed to me, namely "Harbour" made by Gareth Hague. I did research on this particular font and made a website presenting everything I could find.

The website includes a complete overview of all the characters, lowercase and uppercase, special characters and symbols that the font has to offer. There is also information about the origin of the font, where it has taken inspiration from. There is information about the creator of the font and lastly there is a showcase of the font in use in commercial products and publishing. I tried to make the website very clean so that the main focus is on the font itself. All the interesting bits of design in the website are related to or created using characters from the font.

It was a fun website to make and also a good challenge to make it within one day. Usually I really take my time making a website, but having to make it in one day (and succeeding) showed me that I am definitely getting more and more familiar and efficient with coding for the web.
