Interface & Interactie

  • Beschrijving

I was given only one week to code an interactive timeline of any kind. The intended subject for this timeline was to be an existing character or product. I choose to make my timeline about one of my favorite videogames: Outer Wilds. Instead of an actual timeline I made an interactive map of the solar system as it is in the game.

My goal during the making of the project was to rely as little as possible on the use of images. Normally this is something I very quickly gravitate towards, but now I wanted to see how much of the visuals I could create by simply using HTML and CSS. This worked out fairly well as only places were images are used are in the information windows and in the introduction screen. Everything else you see, the starry background, the planets and their orbits were all simply created using code.

It was a lot of fun to work on this project. Everything came together very well and the end product is a simple but effective introduction into the world (or solar sytem) of the Outer Wilds videogame.

The following contains mild spoilers for Outer Wilds:
The reason I choose to make the Outer Wilds solar system is that in the game you are stuck in a short 22-minute timeloop. Within this timeloop certain celestial bodies in the solar sytem undergo large changes. The most prominent one is the sun continuously growing until it gets to heavy and subsequently collapsing in on itself and causing a supernova. This marks the end of another loop. Some other planets also undergo changes and all these changes I've tried incorporating into this simulated version of the solar system, with a shortened version of the timeloop, making this a true timeline.
