Interface & Interactie

  • Beschrijving

Naast de meesterproef was dit mijn favoriete vak van deze minor. Zoals al eerder gezegd is coderen mijn favoriete deel van de opleiding dus het is dan ook niet heel verassend dat dit vak mij goed lag. De opdracht was om (in eigenlijk maar anderhalve week) een interactieve tijdlijn te coderen.

Ik heb ervoor gekozen om een tijdlijn te maken gebaseerd op mijn favoriete game. In deze game bevind je je in een fictief zonnestelsel waarin de alle planeten (en zelfs de zon) snel door grote veranderingen gaan. Dit heb ik geprobeerd na te maken op een kleinere schaal als een soort planetarium. Het venster hierboven is interactief, dus probeer het gerust even uit.

Ik heb zoveel mogelijk code gebruikt om alles te maken en ik wilde zo min mogelijk afhankelijk zijn van afbeeldingen van het internet bijvoorbeeld. Daarom zijn bijna alle visuele elementen enkel gemaakt met HTML en CSS en heb ik verder JavaScript gebruikt voor de interactie en ik ben erg blij met het eindresultaat! Alles is geanimeerd en bijna alles is interactief. Door op de planeten te klikken krijg je meer informatie en elke planeet heeft ook nog eens zijn eigen muziek. Ook zijn er meerdere easter eggs verstopt in de website.
